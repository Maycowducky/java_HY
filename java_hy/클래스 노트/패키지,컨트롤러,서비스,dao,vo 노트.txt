-프로그램 패키지
	- controller
		- 전반적인 작업을 처리하는 곳
		- 카페 목록을 위한 작업
			- 카페 목록 10개를  가져옴
			- 현재 페이지네이션 정보를 가져옴
				- 전체 카페 목록 개수
				- 현재 페이지 정보
	- service
		- 컨트롤러에서 해야하는 작업을 분담
		- 카페 목록 10개를 가져옴
		- 전체 카페 목록 개수 가져옴
		- 현재 페이지 정보를 가져옴
	- dao : data access object
		- 서비스가 DB에 접근할 때 사용
	- vo : value object
		- DB 테이블을 클래스화한 클래스

프로그램 패키지
